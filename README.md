# simple-twitterCLI
## how to use

Ruby need to be in PATH.
```
git clone https://github.com/chao7150/simple-twitterCLI.git
cd simple-twitterCLI
wget http://curl.haxx.se/ca/cacert.pem -O ssl
gem install twitter oauth
ruby app.py
```
